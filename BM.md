> 최종 수정: 2026-04-23

## 핵심 맥락

```
결혼 앱의 특수성
- 사용자 1인당 사용 기간: 약혼 ~ 결혼식 (평균 6~12개월)
- 재구매 없음 → LTV가 짧지만 지불 의향은 매우 높음
- 평균 결혼 비용 3,000~5,000만원 → 앱 비용은 심리적 저항 낮음
- 지인 추천 바이럴 강력 ("나 이거로 결혼 준비했어")
```

## 수익 구조 (레이어별)

### Layer 1. 커플 일시불 유료화 ⭐ 1순위

구독이 아닌 **일시불** — 어차피 1년만 쓰므로 "결혼 준비 기간 전체 이용권" 포지셔닝.

**예상 가격**: 19,900 ~ 29,900원 (커플 계정 공유 포함)

| 기능 | 무료 (Free) | 유료 (Pro) |
|------|------------|-----------|
| 체크리스트 항목 | 30개 | 무제한 |
| 예산 업체 | 10개 | 무제한 |
| 하객 명단 | 50명 | 무제한 |
| 갤러리 사진 | 50장 | 500장 |
| 청첩장 템플릿 | 1개 (기본) | 전체 |
| 커플 메모 | ✅ | ✅ |
| RSVP | ✅ | ✅ |
| 광고 노출 | 추후 검토 | 없음 |

**구현 방향**:
- `couples` 테이블에 `plan` 컬럼 추가 (`free` / `pro`)
- `plan_expires_at` 컬럼 (결혼일 + 3개월 등 만료 기준)
- 결제 게이트웨이: **토스페이먼츠** (Next.js SDK 공식 지원, 카드+카카오페이+네이버페이 한번에)

---

### Layer 2. 프리미엄 청첩장 템플릿 단품 판매

현재 `InvitationTemplates.js` 에 템플릿 렌더러 존재 → 그대로 활용 가능.

**예상 가격**: 템플릿 개당 9,900 ~ 14,900원

- 경쟁사 (모바일청첩장, 포잉 등): 장당 3,000 ~ 15만원
- 차별점: 앱 전체 연동 (하객 관리 + RSVP + 갤러리가 한 앱에)

**구현 방향**:
- `couple.purchased_templates[]` 배열로 구매 템플릿 ID 저장
- 템플릿 선택 시 미구매 → 결제 모달 진입

---

### Layer 3. 업체 B2B 제휴 커미션 (가장 큰 스케일)

현재 `vendors` 테이블 존재 → 제휴 업체 배지·우선 노출 구조로 확장 가능.

**구조**:
1. 커플이 예산 페이지에서 업체 검색 / 추가
2. 제휴 업체는 "파트너" 배지 + 상단 노출
3. 커플이 제휴 업체와 계약 확정 시 → 커미션 수취

| 업체 유형 | 평균 계약액 | 커미션율 | 건당 기대 수익 |
|-----------|------------|---------|--------------|
| 스튜디오 | 200~400만원 | 3~5% | 6~20만원 |
| 드레스샵 | 150~300만원 | 3~5% | 4~15만원 |
| 예식장 | 1,000만원+ | 1~2% | 10~20만원 |
| 허니문 | 300~500만원 | 3~5% | 9~25만원 |

> 커플 1쌍 평균 3~4개 업체 계약 → **커플당 기대 커미션 20~50만원**

**초기 운영**: 자동화 전에 수동으로 업체 연락 → 계약 → 정산
**구현 방향**: 업체 어드민 페이지 별도 개발 필요

---

### Layer 4. 갤러리 이벤트 패키지 (단품)

결혼식 당일 하객 사진 공유 기능 이미 존재 (`photo_events`, `guest_photos` 테이블).

| 플랜 | 가격 | 용량 | 보관 |
|------|------|------|------|
| 기본 | 무료 | 50장 | 90일 |
| 결혼식 패키지 | 9,900원 | 500장 | 6개월 |
| 프리미엄 | 19,900원 | 무제한 | 1년 + zip 다운로드 |

## 광고 (보류)

### 현황 및 결론

| 방식 | 기술적 가능 여부 | 비고 |
|------|----------------|------|
| Google AdSense | 브라우저 버전만 ✅ | WebView 앱 내 사용은 Google ToS 위반 |
| Google AdMob | ✅ | `@capacitor-community/admob` 플러그인 필요, 네이티브 SDK 추가 공수 있음 |
| 카카오 애드핏 | ✅ | WebView 허용, 설치 비교적 쉬움 |

### 보류 이유

- 결혼 준비 앱에 배너 광고 → UX/프리미엄 감 심각하게 훼손
- 한국 모바일 CPM 500~1,500원 수준 → DAU 500명 기준 월 2~5만원 (의미 있는 수익 아님)
- **"광고 없음"을 Pro 플랜 혜택으로 포지셔닝하는 게 더 효과적**

### 재검토 조건

- DAU 5,000명 이상 도달 시 AdMob 배너 검토 (무료 사용자 대상만)
- 위치: 하단 네비 위 고정 배너 (BottomNav와 겹치지 않게 `pb-24` 조정 필요)

### 월 신규 커플 100쌍 기준

```
일시불 전환율 40%  → 40쌍 × 29,900원        = 1,196,000원
청첩장 템플릿 30%  → 30쌍 × 12,000원         =   360,000원
업체 커미션 (초기) → 5건  × 100,000원         =   500,000원
─────────────────────────────────────────────────────────
월 합계 ≈ 206만원 / 연 합계 ≈ 2,500만원
```

```
청첩장 템플릿 30%  → 150쌍 × 12,000원         = 1,800,000원
업체 커미션        → 30건  × 100,000원          = 3,000,000원
─────────────────────────────────────────────────────────
월 합계 ≈ 1,078만원 / 연 합계 ≈ 1.3억원
```
