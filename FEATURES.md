| 앱 | 주요 BM | 부수 수익 | 현재 상태 |
|----|---------|-----------|----------|

| **웨딩북** | 업체 중개 수수료 + B2B ERP 솔루션 | 캐시백 환원, 오프라인 공간, 광고 노출 | 매출 43.8억/년, 시리즈C 투자 |

### 💡 각 BM 상세 분석

#### 호닌 — 클린 플랫폼 (현재 무수익)
- "광고도 플래너 영업도 없는 앱" 표방 → 초기 신뢰 확보 전략
- 인앱결제 없음, 광고 없음 → 사용자 규모 먼저 키우는 단계
- **예상 수익화 경로**: 프리미엄 구독, 업체 디렉토리, 커머스 연동

#### 웨딩북 — O2O 플랫폼 (업계 1위)
- **핵심**: B2B ERP (수도권 웨딩홀 40%, 드레스샵 80% 점유) → 업체 락인
- 소비자에게 캐시백 최대 10% 제공 → 업체 중개 수수료로 충당
- 2026년 1월 AI 웨딩플래너 기능 출시
- 누적 투자 158억원, 수도권 신부 60% 사용

#### 요즘웨딩 — 오프라인 연계 플랫폼
- "드레스랜드": 드레스 피팅 + 헤어메이크업 + 스튜디오 체험 원스톱 공간
- 디지털 앱보다 오프라인 공간이 핵심 수익원

### 🎯 Ourday BM 시사점

```
단계별 수익화 로드맵 (추천)
─────────────────────────────
1단계 (현재 ~ 사용자 1만명)
  → 완전 무료: 신뢰 확보 & 사용자 획득 (호닌 전략)

2단계 (1만명 이상)
  → 프리미엄 구독 ₩4,900/월 or ₩39,000/년
    - 무제한 하객 (무료: 50명)
    - QR 하객 사진 업로드 (무료: ❌)
    - 모바일 청첩장 무제한 열람
    - 영수증·계약서 무제한 사진 첨부

3단계 (커플 5천쌍 이상)
  → 업체 디렉토리 + 중개 수수료 (웨딩북 모델)
  → 업체 프리미엄 노출 광고
```

## QR 코드 하객 사진 업로드 기능 분석

> 글로벌 시장에서 빠르게 성장 중인 기능. 한국에서는 아직 공백 영역.

### 작동 원리
1. 커플이 앱에서 이벤트 생성 → 고유 URL + QR 코드 자동 발급
2. QR 코드를 테이블 카드·청첩장·식장 디스플레이에 인쇄
3. 하객: 스마트폰 카메라로 QR 스캔 → 브라우저 자동 열림
4. **앱 다운로드 없음, 계정 없음** → 바로 사진 선택 후 업로드
5. 커플 갤러리에 실시간 수집 + 라이브 슬라이드쇼 표시 가능

### 글로벌 경쟁 서비스 가격 비교

| 서비스 | 무료 플랜 | 유료 가격 | 결제 방식 |
|--------|-----------|-----------|-----------|
| **Fotify** | 50장, 7일 | ₩39,900 ~ ₩69,000 | 이벤트당 일회성 |
| **Wedibox** | 50장 | $49 ~ $99 | 이벤트당 일회성 |
| **Lense** | 있음 | ~$69.99 | 이벤트당 일회성 |
| **Kululu** | 있음 | ~$99 | 이벤트당 일회성 |
| **Waldo Photos** | 있음 | $9.99/월 | 구독 |

> **핵심 인사이트**: 한국 웨딩 앱 중 이 기능을 제공하는 서비스 없음 → **Ourday 차별화 포인트**

### Ourday 구현 계획 (P1 추가)
- 무료: QR 생성 + 사진 50장 + 7일 보관
- 프리미엄: 무제한 사진 + 영구 보관 + 라이브 슬라이드쇼

## 구현 순서 추천

```
1단계 (MVP 완성)
  → 하객 관리 + 축의금
  → 업체 관리 (계약 현황)
  → 캘린더 뷰

2단계 (차별화)
  → QR 하객 사진 업로드 ← 국내 공백, 차별화 핵심
  → 커플 채팅 (Supabase Realtime)
  → 모바일 청첩장

3단계 (수익화)
  → 프리미엄 구독 도입 (QR 무제한, 사진 무제한)
  → 알림 시스템 (Web Push)
  → 소셜 로그인 (카카오)
  → PDF 내보내기
```
