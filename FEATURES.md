## P0 — 핵심 누락 기능 (경쟁 앱 공통 필수)

### 🎊 하객 관리
- [ ] 하객 명단 등록 (이름·연락처·신랑측/신부측·관계)
- [ ] 하객 수 집계 (신랑측/신부측/합계 실시간 표시)
- [ ] 식사 수량 자동 계산 (하객 수 기반)
- [ ] 축의금 수입 기록 (하객별 금액 입력)
- [ ] 축의금 합계 vs 지출 합계 비교 (손익 요약)

### 📅 캘린더 뷰
- [ ] 월간 달력 화면 (체크리스트 마감일 시각화)
- [ ] 업체 미팅 일정 등록 및 표시
- [ ] 잔금 납부일 등록 및 D-day 표시
- [ ] 일정 알림 (브라우저 Push Notification)

### 🏢 업체 관리
- [ ] 업체 정보 등록 (종류·업체명·담당자·연락처·계약금·잔금·납부일)
- [ ] 업체 후보 비교표 (웨딩홀 A vs B vs C)
- [ ] 계약 상태 태그 (미계약 / 계약완료 / 잔금대기)
- [ ] 계약서·영수증 사진 첨부 (Supabase Storage)

## 구현 순서 추천

```
1단계 (MVP 완성)
  → 하객 관리 + 축의금
  → 업체 관리 (계약 현황)
  → 캘린더 뷰

2단계 (차별화)
  → 커플 채팅 (Supabase Realtime)
  → 모바일 청첩장

3단계 (고도화)
  → 알림 시스템 (Web Push)
  → 소셜 로그인 (카카오)
  → PDF 내보내기
```
